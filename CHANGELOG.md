# Changelog

## 1.0.0 (2026-01-19)

### Features

* Created comprehensive MapColonies app boilerplate with:
  * Vite + React + TypeScript setup
  * React Router for client-side routing
  * i18n support for Hebrew and English
  * Material-UI theming system
  * Cesium integration for 3D maps
  * Runtime environment variable configuration via confd
  * Helm charts for Kubernetes deployment
  * Docker containerization
  * Release-please automation
  * ESLint and Prettier for code quality
  * Clean architecture with no business logic or GraphQL
